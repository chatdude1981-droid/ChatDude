ChatDude

Privacy Notes

This is a lightweight early-stage privacy placeholder for demos and small hosted communities. It is not legal advice.

What ChatDude stores

Registered accounts store a username, display name, password hash, preferences, rooms, and message history. Guest sessions are lighter and may be more temporary.

Realtime activity

Presence, messages, typing, and media state are transmitted through the realtime backend so rooms and direct messaging work properly.

Media

Camera and call features use browser WebRTC where available. Audio/video access is requested from the browser directly and should only activate after user permission.

Early-stage limitation

If the app is deployed without stable Postgres persistence, data can be lost on redeploys. Production use should configure a durable database.
